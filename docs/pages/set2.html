Set 2: Failing Computations

This set deals with programming patterns that arise when dealing with computations that might fail. Failing computations come up all the time in the real world and PureScript can provide great tools for dealing with them.

  1. The Maybe type
  2. Build a library of things that can fail
  3. Chains of failing computations
  4. Generalizing chains of failures
  5. Chaining variations
  6. Tailprod
